> 讲一下http和https的区别

http

超文本传输协议，应用最为广泛的传输协议，以明文发送信息，这样不安全。

https

认为是http的安全版，在此基础上加上了ssl协议，加在了传输层和应用层之间，对通信数据进行加密，验证身份码，更加安全。

| 区别         | http   | https                               |
| ------------ | ------ | ----------------------------------- |
| 连接方式不同 | 端口80 | 端口443                             |
| 传输方式     | 明文   | 加密                                |
| 必要条件     |        | 需要申请ca证书                      |
| 速度         | 快     | 需要ssl握手，有更复杂的安全过程，慢 |
